# 🏀 NBA Spark ETL Dashboard

### Created for the AQX Sports Analytics Hackathon

An end-to-end NBA analytics project that extracts player statistics from the NBA API, transforms and analyzes the data using PySpark, and visualizes insights through an interactive Streamlit dashboard.

---

## 🚀 Features

### 🔍 Player Search
Search for any NBA player by name and instantly retrieve their season statistics.

### 📊 Season Averages
View key performance metrics including:

- Points Per Game (PPG)
- Rebounds Per Game (RPG)
- Assists Per Game (APG)

### 📈 Performance Analytics
Analyze player trends using:

- Rolling 5-Game Average Points
- Season Summary Statistics
- Consistency Metrics

### 🏀 Complete Game Logs
Explore full game-by-game performance data

### ⚔️ Player Comparison
Compare two NBA players side-by-side across multiple statistical categories

### 📉 Interactive Visualizations
Visualize player performance using dynamic Plotly charts and graphs

### ⚡ PySpark ETL Pipeline
Leverages PySpark for:

- Data Extraction
- Data Transformation
- Statistical Aggregation
- Rolling Window Calculations
- Analytics Processing
